Alessia La Rocca
till June 29th 2007

On 29th June 2007, the first iPhone was presented; since that day, the way people collect their memories has changed. With the arrival of social media and the increasingly powerful smartphone cameras, the habit of taking photographs with roll cameras to be stored in paper albums has been abandoned.
In this project, I have collected and digitised my photographs found in these albums from 1998, my year of birth, until 2007.
I reinterpreted them in a more contemporary way by giving them a caption as one would do now by posting them on social networks.

Archive

mouseenter: preview, previous, next

The preview (and consequently the previous and next features) is created with a mouseenter in the element. I started by understanding how to handle the elements from the json and the final effect causes the element to be first displayed as a caption and then showing the image.

filters

Filters manage the tags attributed to the image, in this case the people in the picture are categorised.

sort

The sort function sorts the items; you can manage the year from oldest to newest and the place where the photo was taken in alphabetical order or the reverse.

keypress: shuffle

The input for the shuffle button is done with the keypress, in this case using the letter 'g'.

search

The search bar inspects the elements inserted within the caption.

mousemove: x axis

A mousemove function has been added that selects the image class in css and changes its position on the x-axis. Vertically, there was no need to update the y-position as it already depends on the mouseenter in the items.

year digits

It was important to give the year data in the json file the exact date so that the sort would sort the photos correctly, but visually this full data was not needed as only the first four digits of the year are shown. Therefore, when inserting the contents, a function had to be added to show only these first digits.

overflow and blend mode

The header elements and filters are fixed at the scroll of the page so that the tag and sort can be managed at any time. A transparency blend was added to separate the elements from the background.
